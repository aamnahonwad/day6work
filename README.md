📚 Java Collection Framework 

This repository contains programs based on the Java Collection Framework.
The programs demonstrate the use of:

ArrayList

LinkedList

HashSet

LinkedHashSet

TreeSet

🔹 Topics Covered
📌 ArrayList Programs

Store and print student marks

Calculate average

Search student name using contains()

Update employee salary using set()

📌 LinkedList Programs

Bus stop queue using addFirst() and addLast()

Remove first customer using removeFirst()

Add emergency patient using addFirst()

Print first and last task using getFirst() and getLast()

Remove last task using removeLast()

📌 Set Programs
1️⃣ HashSet

Remove duplicate student IDs

Store unique city names

2️⃣ TreeSet

Store and automatically sort student roll numbers

3️⃣ LinkedHashSet

Maintain insertion order of book titles

Remove duplicate entries

4️⃣ Compare Set Types

Compare HashSet, LinkedHashSet, and TreeSet

Understand differences in:

Order

Sorting

Duplicate handling

Size comparison
